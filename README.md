A streamlined music streaming application crafted for casual listeners. Sangeet Lite is lightweight and easy to set up, letting you enjoy music without unnecessary complexity.

## Features

* **Simple Setup** - Minimal installation steps to get you started fast
* **Lightweight** - Designed for casual music listeners who want simplicity and efficiency
* **Cross-Platform** - Compatible with Windows, Mac, and Linux systems

## Prerequisites

* Python 3.6 or higher
* Internet connection for streaming

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/robotic-s/sangeet-lite.git
cd "sangeet radio plus"
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

> **Note:** If your system uses multiple Python versions, you may need to use `pip3` instead of `pip`.

## Usage

Start Sangeet Lite using either of these commands:

```bash
python radio.py
# OR
python3 radio.py
```

The application will automatically begin streaming music once launched.
